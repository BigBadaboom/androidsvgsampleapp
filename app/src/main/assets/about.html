About this app

The purpose of this app is to demonstrate the abilities of the AndroidSVG library, and provide some real world examples of how to use the library to include SVG images in your Android applications.

For more information about AndroidSVG, and documentation, visit:

http://bigbadaboom.github.io/androidsvg/

or on GitHub:

https://github.com/BigBadaboom/androidsvg

The source code to this app is available at:

https://github.com/BigBadaboom/androidsvgsampleapp.
